# Sistema Web de Gerenciamento de Playlists

Aplicação web full stack desenvolvida como projeto acadêmico, com foco em integração entre frontend, backend e persistência de dados em banco relacional.

## Objetivo

Desenvolver uma aplicação completa para gerenciamento de playlists musicais, aplicando conceitos de:
- CRUD
- Modelagem de banco de dados
- Autenticação de usuários
- Organização modular de código

## Funcionalidades

- Cadastro e login de usuários
- Controle de sessão
- Criação, edição e exclusão de playlists
- Associação de músicas às playlists
- Sistema de favoritos
- Persistência de dados em MySQL
- Separação entre frontend e backend

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- PHP
- MySQL
- Git
